Spring Boot Kafka Examples

Collection of Spring Boot Kafka projects demonstrating different serialization techniques.

Overview

This repository demonstrates multiple approaches for publishing and consuming Kafka messages using Spring Boot. Each project is independent and focuses on a different serialization strategy.


Project Description Serialization
Spring Boot Kafka Simple Producer & Consumer using String and JSON messages. String / JSON
Spring Boot Kafka Protobuf Kafka integration using Google Protocol Buffers. Protocol Buffers
Spring Boot Kafka Avro Kafka integration using Apache Avro. Apache Avro

docker compose up -d

mvn clean install
